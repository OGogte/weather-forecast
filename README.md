# Weather Forecasting App

Deployed link - https://master--weather-forecast-app-symplora.netlify.app/

Steps to run locally

1. ```bash
   git clone https://github.com/OGogte/weather-forecast-app.git
   ```
2. ```bash
   cd "./weather-forecast-app"
   ```
3. ```bash
   npm i
   ```
4. ```bash
   npm run dev
   ```
